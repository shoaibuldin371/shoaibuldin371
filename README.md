## Featured Project

### AI Powered E Commerce Chatbot

**Final Year Project | Oct 2025 to May 2026**  
**Tech Stack:** Next.js, Python, NLP, Google Colab

An AI powered chatbot integrated into a custom online store. The system helps users search products using simple natural language queries such as:

> black shoes under 2000

### Key Features

- Built a chatbot for natural language based product search
- Trained a custom NLP model on product data using Google Colab
- Extracted product category, color, and price range from user queries
- Improved product discovery by reducing dependency on fixed filters
- Added product recommendation logic for better user experience
- Applied practical concepts of e commerce, AI, NLP, and web development
